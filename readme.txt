== Description ==

The quick way to integrate with Affilio platform. Auto integrate products, categories and orders with Affilio Platform.

== Installation ==

Upload the Affiliate plugin to your blog, activate it, and then enter your Affiliate.ir API key.

1, 2, 3: You're done!
